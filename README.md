# Shukatsu MyPage Autofill

就活の企業マイページ登録フォームに、名前・住所・電話番号・学校名などをまとめて入力するための Chrome / Arc 用拡張機能です。

登録ボタン、送信ボタン、次へボタンは自動では押しません。入力後は必ず自分で内容を確認してから進んでください。

## できること

- 氏名、フリガナ、メールアドレス、電話番号、住所などを保存できます。
- 学校情報、資格、TOEIC、希望職種、希望勤務地なども保存できます。
- 高校名、高校よみがな、高校の入学・卒業年月、ローマ字氏名も保存できます。
- 規約同意、個人情報同意、国籍、在留資格、就労資格、認証コードなども、保存した内容に合わせて入力できます。
- ログインID、海外住所チェック、紹介者名、勤務経験ブランド・店舗・社員番号、転勤確認回答なども事前に保存できます。
- 参加希望イベント、イベント希望日程などの選択式項目にも対応しています。
- i-web、sonar ATS、AOL、e2R pro などの企業マイページ系フォームで、近い項目を見つけて入力します。
- フォームに何個入力できそうか確認できます。
- 未入力の必須欄が残っていないか確認できます。
- 入力後レビューで、入力された項目、残っている必須欄、未判定欄、スキップ項目を分けて確認できます。
- ページ上の入力欄を色付けして、どこを自動判定しているか見られます。
- 判定できなかった入力欄の例も確認できます。
- 判定できなかった入力欄や未入力の必須欄をコピーして、対応依頼しやすくできます。
- 保存データを書き出したり、別のブラウザに読み込んだりできます。
- ブラウザに保存した個人情報をボタンで削除できます。
- 「新規登録」「エントリー」などの入口っぽいボタン候補を探せます。

## 対応を強めているサイト

### 実際の登録フォームで入力確認済み

- i-web
- sonar ATS / snar.jp
- AOL / アクセスオンライン（job.axol.jp 系）
- e2R pro
- HITO-Link 新卒特化版
- HRMOS採用 新卒版
- My CareerBox / OpenES

### 対応ルールを入れているサイト

- JobSuite FRESHERS

JobSuite FRESHERS は、誰でも開ける公開中の新規登録フォームを見つけられていないため、実フォームでの入力確認はまだです。

## 先に知っておくこと

- マイナビ、リクナビなどの就活ナビサイト用ではありません。
- ただし、My CareerBox / OpenES は対応対象に入れています。
- 企業ごとの「マイページ登録フォーム」で使う想定です。
- 全サイトで100%完璧に入るわけではありません。サイトごとに項目名が違うため、最後は目で確認してください。
- 個人情報は自分のブラウザ内に保存されます。外部サーバーには送りません。
- `保存データを書き出す` で作ったファイルには個人情報が入ります。人に送らないでください。
- 規約同意や個人情報同意は、自分がチェックをオンにした場合だけ入力されます。同意内容は必ず自分で読んでください。
- 志望動機、自己PR、ガクチカなどの文章回答は自動入力しません。
- 会社ごとの固有すぎる設問は無理に入力しません。入力後レビューで残った欄を確認してください。

## インストール方法

### 1. ファイルをダウンロードする

1. GitHub のこのページを開きます。
2. 右側、またはページ内の `Releases` を押します。
3. 一番新しいリリースを開きます。
4. `Assets` の中にある `chrome_shukatsu_autofill.zip` を押します。
5. ダウンロードされた ZIP ファイルを右クリックして、`すべて展開` を押します。
6. 展開したフォルダの中に `manifest.json` があることを確認します。

`Source code (zip)` ではなく、できれば `chrome_shukatsu_autofill.zip` を使ってください。その方が迷いにくいです。

### 2. Chrome に入れる

1. Chrome を開きます。
2. アドレスバーに `chrome://extensions/` と入力して Enter を押します。
3. 右上の `デベロッパー モード` をオンにします。
4. `パッケージ化されていない拡張機能を読み込む` を押します。
5. さっき展開したフォルダを選びます。
6. `Shukatsu MyPage Autofill` が表示されたら完了です。

### 3. Arc に入れる

1. Arc を開きます。
2. アドレスバーに `arc://extensions/` と入力して Enter を押します。
3. 右上の `デベロッパー モード` をオンにします。
4. `パッケージ化されていない拡張機能を読み込む` を押します。
5. さっき展開したフォルダを選びます。
6. `Shukatsu MyPage Autofill` が表示されたら完了です。

## 使い方

### 1. 最初にプロフィールを保存する

1. Chrome または Arc の右上にある拡張機能アイコンを押します。
2. `Shukatsu MyPage Autofill` を開きます。
3. 名前、フリガナ、メール、住所、電話番号、学校名などを入力します。
4. `保存` を押します。

保存した内容は、次回から同じブラウザで使えます。

### 2. 企業マイページで入力する

1. 企業のマイページ登録フォームを開きます。
2. 拡張機能アイコンから `Shukatsu MyPage Autofill` を開きます。
3. `入力欄チェック` を押します。
4. 入力できそうな項目数と、未入力の必須欄が表示されます。
5. 必要なら `入力欄を表示` を押します。
6. ページ上で、緑は判定済み、茶色は未判定、赤は未入力の必須欄として表示されます。
7. 問題なさそうなら `現在ページに入力` を押します。
8. `入力後レビュー` で、入力された項目と残っている必須欄を確認します。
9. 入った内容をページ上で全部確認します。
10. 間違いがなければ、自分で登録ボタンや次へボタンを押します。

色付けを消したいときは `表示を消す` を押してください。

### 3. 保存データをバックアップする

別のブラウザや別のパソコンでも同じ内容を使いたい場合は、保存データを書き出せます。

1. 拡張機能を開きます。
2. `保存データを書き出す` を押します。
3. `shukatsu-autofill-profile.json` がダウンロードされます。
4. 別のブラウザで使うときは、拡張機能を開いて `保存データを読み込む` を押します。
5. 書き出した JSON ファイルを選びます。

この JSON ファイルには、名前、住所、メール、パスワードなど、入力した内容が入ります。扱いには注意してください。

### 4. 保存情報を消す

使わなくなった場合や、共有パソコンで使った場合は、拡張機能内の `保存情報を消す` を押してください。

拡張機能そのものを削除したい場合は、このページ下の `削除方法` も見てください。

### 5. 入らない項目を共有する

1. 登録フォームを開きます。
2. `入力欄チェック` を押します。
3. `未判定をコピー` を押します。
4. コピーされた内容を、対応してほしい人に送ります。

URL、サイト名、未判定の入力欄、未入力の必須欄がまとまってコピーされます。

## Arc で拡張機能が見つからないとき

1. Arc の上の方にある拡張機能のボタンを押します。
2. 一覧から `Shukatsu Autofill` を探します。
3. 見つからない場合は、`arc://extensions/` を開いて有効になっているか確認してください。

## よくある困りごと

### 入力されない

- そのページが本当に登録フォームか確認してください。
- ページを一度再読み込みしてから、もう一度試してください。
- `入力欄チェック` で検出数が 0 の場合、そのサイトの項目名にまだ対応できていない可能性があります。

### 一部だけ入らない

- サイトによって項目名や作りが違うため、一部だけ入らないことがあります。
- 入らなかったところは手で入力してください。
- `入力欄を表示` を押すと、未判定や未入力の必須欄を見つけやすくなります。
- `入力後レビュー` に残っている必須欄が出るので、そこから確認してください。
- 何度も使うサイトなら、そのページのURLやスクショを見せてもらえれば対応を追加できます。

### 間違った場所に入った

- `上書きする` のチェックを外してから使うと、すでに入力済みの欄は触りません。
- 入力後は必ず目で確認してください。

### 登録ボタンが押されない

これは正常です。間違って応募・登録しないように、送信系のボタンは自動で押さないようにしています。

## 入力確認メモ

公開されている登録フォームで、送信せずに入力だけ確認しています。

2026年6月2日時点で、以下の公開フォームはすべて未判定0件、未入力必須0件まで確認しています。

- i-web: 33項目を判定、33件入力
- sonar ATS / snar.jp: 33項目を判定、36件入力。ログインID欄も対応
- AOL / アクセスオンライン: 84項目を判定、69件入力。研究専門性、学校系連動プルダウンも含めて確認
- e2R pro: 46項目を判定、40件入力。学校検索が必要な学校・学部・学科、住所分割、学科系統も対応
- HITO-Link 新卒: 同意ページから入力フォームへ進み、59項目を判定、52件入力。性別、生年月日、都道府県、郵便番号ハイフンなし、学校区分、文理区分、特殊プルダウンも含めて確認
- HRMOS採用 新卒版: 26項目を判定、22件入力。学校の在籍期間と卒業予定確認も対応
- My CareerBox / OpenES: 9項目を判定、9件入力。メール認証コードも欄としては対応

## 安全のための注意

- 本番の応募フォームでは、入力後に必ず内容を確認してください。
- パスワードを保存する場合は、他人と同じパソコンで使わないでください。
- 書き出した JSON ファイルには個人情報が入るため、共有しないでください。
- 共有パソコンで使った場合は、`保存情報を消す` を押してから拡張機能を閉じてください。
- 使わなくなったら、拡張機能画面から削除できます。

## 削除方法

1. `chrome://extensions/` または `arc://extensions/` を開きます。
2. `Shukatsu MyPage Autofill` を探します。
3. `削除` を押します。
